# water-wave
一个创建点击后产生水波纹效果的React组件

### 安装
    $ npm install --save water-wave

### 使用
```jsx
    import WaterWave from 'water-wave';
    import 'water-wave/style.css';

    const MyComponent = () =>
        <button className="btn">
            按钮
            <WaterWave color="#fff" duration=800 />
        </button>;
```
